# 🛒 Tec-View P v1

**Tec-View P v1** es una aplicación web desarrollada con **React** y **Firebase**, diseñada para simular un sistema de gestión de tienda con autenticación de usuarios, control de acceso por roles y administración de productos.

El proyecto fue desarrollado con un enfoque en la experiencia del usuario, la seguridad de acceso y la administración eficiente de la información mediante una base de datos en la nube.

# 📋 Características

## 🔐 Sistema de autenticación

- Registro de nuevos usuarios.
- Inicio de sesión mediante credenciales.
- Validación de autenticación.
- Cierre de sesión seguro.

---

## 👥 Gestión de roles

La aplicación implementa un sistema de autorización basado en roles.

Actualmente existen dos perfiles principales:

### 👤 Cliente

Los usuarios con este rol pueden:

- Visualizar los productos disponibles.
- Buscar productos.
- Filtrar productos.
- Consultar información y precios.

### 🛠 Administrador

Los administradores tienen acceso al panel de gestión donde pueden:

- Administrar el catálogo de productos.
- Agregar nuevos productos.
- Editar información existente.
- Eliminar productos.
- Gestionar la información almacenada en Firebase.

---

## 🛍 Gestión de productos

El sistema permite:

- Visualización dinámica de productos.
- Búsqueda por nombre.
- Filtros para facilitar la navegación.
- Consulta de precios y detalles.
- Actualización automática de la información desde Firebase.

---

## 🔥 Base de datos

La aplicación utiliza **Firebase** como plataforma Backend-as-a-Service (BaaS).

Se implementó:

- Firebase Authentication
- Cloud Firestore
- Gestión de usuarios
- Almacenamiento de datos en tiempo real
- Control de acceso según el rol del usuario
